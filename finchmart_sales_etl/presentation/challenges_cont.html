Incorrect CountDistinct Syntax
Problem: Used count(col("customer_id").distinct()) incorrectly
Error: TypeError: 'Column' object is not callable
Solution:
Replaced with separate distinct().count() or countDistinct() aggregation. Lesson: Understand PySpark API nuances—distinct() returns DataFrame.
5
Enrichment Increased Record Count from 600 to 950
Problem: Left join with product table increased records unexpectedly
Root Cause: Product table contains duplicate product_ids with different attributes (multiple variants per product)
Solution:
Accepted duplication for demo; documented deduplication strategy for production using window functions to keep one record per product_id. Lesson: Always validate reference data quality before joins to prevent unexpected record multiplication.
